# Cancer AI System

A hospital-grade AI system for comprehensive cancer analysis using deep learning.

## Models

### Model 1 — Cancer Detection
- Task: Binary classification (Cancer vs No Cancer)
- Architecture: DenseNet121
- Dataset: IDC Breast Histopathology (277,524 images)
- AUC-ROC: 0.9432 | Sensitivity: 94.64%
- Demo: https://huggingface.co/spaces/Abdulhaque/cancer-detection-app

### Model 2 — Cancer Type Classification
- Task: 5-class classification
- Architecture: ConvNeXt-Base
- Dataset: LC25000 (25,000 images)
- Accuracy: 94.2% | AUC: 0.9942
- Demo: https://huggingface.co/spaces/Abdulhaque/cancer-type-classification-app

### Model 3 — Cancer Stage Prediction
- Task: Stage I, II, III, IV prediction
